# Tugas-Akhir-Judul-2-JK

Ini adalah penjelasan mengenai konfigurasi dasar dari device switch dan router dan bagaimana cara kerja kedua device ini dalam membagi jaringan yang terhubung,
pada video ini ketika ping dimasukkan kepada kedua PC yang tersambung melalui switch dan router maka perintah ping yang digunakan untuk uji konektivitas tidak
akan menghasilkan reply yang dikarenakan kedua PC masih berada pada jaringan berbeda yang dipisahkan oleh Router namun ketika router itu sendiri sudah diberikan
alamat IP dan default gateway sebagai alamat untuk mencapai jaringan di luarnya maka ping berhasil dan kedua PC dapat bertukar informasi
